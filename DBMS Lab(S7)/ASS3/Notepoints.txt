*Operating xampp in linux based OS
Opening xampp/lampp in any linux based OS:
open terminal from anywhere(assumed lampp already installed)
cd /opt/lampp
ls
sudo ./manager-linux-x64.run
starts xampp application
click start all button in application
Now open any browser and type localhost/phpmyadmin
create a database with some name and paste whole database.sql file in it and run it
